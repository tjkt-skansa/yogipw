Pembuatan Aplikasi Automasi Barcode berbasis Excel

Pada bulan ketiga ini, saya diberikan tugas untuk membuat sebuah aplikasi automasi barcode berbasis Excel. Aplikasi ini bertujuan untuk memudahkan proses pendataan box arsip. Aplikasi ini dibuat sederhana, yaitu dengan menghasilkan barcode dari data yang ada di Google Sheets. Barcode tersebut kemudian dapat dicetak dan ditempel pada barang yang akan didata. Selanjutnya, barcode bisa dipindai menggunakan aplikasi pemindai, dan hasil scan otomatis tersimpan sebagai data di Google Drive.

Contoh cara pembuatan aplikasi di appsheets

  • Siapkan Data di Excel/Google Sheets: Buat file di Google Drive dengan tabel rapi dan header di baris pertama (misalnya: Kode Barang, Nama Barang, Jumlah, Lokasi).
  • Hubungkan Data ke AppSheet: Buka AppSheet, pilih Make a new app → Start with your own data, lalu hubungkan ke file Excel/Google Sheets Anda.
  • Atur Tampilan Aplikasi: Tambahkan Form View untuk input data, Table/List View untuk melihat data, dan Detail View jika perlu.
  • Tambahkan Fitur Tambahan: Aktifkan Barcode/QR Code Scanner di kolom tertentu, atau tambahkan workflow sesuai kebutuhan.
  • Jalankan dan Uji Aplikasi: Simpan aplikasi, buka di smartphone atau browser, lalu uji coba input dan pastikan data tersimpan otomatis di Google Sheets.
