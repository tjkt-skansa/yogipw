Cara Penginstalan Driver Printer

Langkah Pertama, pergi ke website brand printer

Karena saya menggunakan printer canon g3020, maka saya pergi ke website canon di https://www.canon.co.id/support/consumer_products/products/printers/inkjet/pixma_g_series/pixma-g3020.html?type=drivers&language=&os=WINDOWS%2010%20(64-bit)
Langkah Selanjutnya
  1. Download driver printer sesuai tipe dan sistem operasi.
  2. Buka file installer yang telah diunduh.
  3. Ikuti petunjuk instalasi hingga muncul pilihan.

    Pilih metode koneksi printer, bisa via USB atau WiFi.
  4. Dan tunggu proses instalasi hingga selesai.

  5. Restart komputer jika diperlukan.
  6. Pastikan printer terdeteksi dan siap digunakan.
